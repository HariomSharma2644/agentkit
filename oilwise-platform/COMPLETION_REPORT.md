## 🚀 Ready to Use

### Quick Start
```bash
git clone https://github.com/oilwise/platform.git
cd oilwise-platform
cp .env.example .env
docker-compose up -d
```

### Access Points
- Backend API: http://localhost:3000
- AI Engine: http://localhost:5000
- Web Dashboard: http://localhost:3001
- MQTT Broker: localhost:1883
